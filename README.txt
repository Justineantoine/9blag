PROJET 9blag:
ANTOINE Justine, CHAMBON Léa, MANCINI Matthieu, PICARD Lauren

Ce projet est une galerie d'images inspirée du fameux site 9gag mais made in France.
Ce site a été codé en HTML/CSS et Python grâce au module Flask.

Sur ce site, vous pouvez :
	- parcourir les images : il suffit de scroller
	- les liker ou les disliker (à l'infini mais c'est pas grave) : en cliquant sur les boutons adéquats, le nombres de likes et dislikes par photos sont précisés
	- faire des recherches par mots clés : dans la barre de recherche en haut à droite
	- ajouter une image : dans la barre prévue à cet effet en haut. Une fois publier, vous serez renvoyer vers la page Frais pour pouvoir directement liker votre image !

Vous trouverez deux onglets sur le site :
	- Frais (c'est fraiment vrai) : affiche les dernières nouveautés de 9blag
	- Tendance (le disco) : affiche les images en fonction de leurs nombres de likes et dislikes
	
Améliorations possibles, il y en a énormément mais notamment :
	- revoir tout le CSS
	- Pouvoir liker ou disliker qu'une seule fois par utilisateur
